Developed an AI-powered tweet generation tool that creates brand-specific tweets by analyzing brand voice and using prompt-based generation.

Built the application using Python and Streamlit, enabling users to input a brand name and automatically generate multiple tweet variations.

Implemented prompt engineering and OpenAI-based text generation to maintain tone, style, and engagement suitable for social media marketing.

Designed a simple interactive UI where users can test tweet generation for different brands and explore variations in tone and messaging.
